COMP 115 PROJECT-1 (ARTISTIC DRAWING)

BY- JAPNEET KAUR

THE PROJECT CREATES A WONDERFUL LANDSCAPE WITH GREEN GRASS AND YELLOW SUN ON THE TOP. IT ALSO HAS CLOUDS IN IT TO SHOWCASE WEATHER ALONG WITH FLYING BIRDS AS A
CHERRY ON TOP. THE DRAWING ALSO INCLUDED A COLORFUL RAINBOW TO IT. 

THE GOAL WAS TO PRACTICE ARTISTIC DRAWING USING TURTLE MODULE 

THE PROJECT CONTAINS:-
* RAINBOW WITH COLORFUL ARCS
* GREEN GRASS AT THE BOTTOM
* SUN AT THE TOP
* CLOUDS ALONG THE SUN
* FLYING BIRDS BETWEEN CLOUDS AND THE SUN

  CONCEPTS USED IN THE PROJECT:-
  * IMPORTING TURTLE TO SHOW GRAPHICS
  * USING FOR LOOPS 
  * DEFINING FUNCTIONS TO AVOID REPITITION
  * SETTING POSITION FOR X AND Y COORDINATES FOR THE TURTLE MOVEMENT
  * USING LIST OF COLORS FOR THE RAINBOW
